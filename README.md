# Stack Stories

**Stack Stories** is a full-stack blogging platform built using the MERN stack: MongoDB, Express.js, React, and Node.js. This project features secure JWT-based authentication and offers full CRUD functionality for blog posts. It’s designed with a responsive layout to provide an optimal user experience on both desktop and mobile devices.

## Features

- **JWT-based Authentication:** Secure login and registration.
- **CRUD Operations:** Create, read, update, and delete blog posts.
- **Responsive Design:** Enjoy a seamless experience on any device.
- **Modern Tech Stack:** Built with MongoDB, Express.js, React, and Node.js.

## Prerequisites

- [Node.js](https://nodejs.org/) (v12 or higher recommended)
- [npm](https://www.npmjs.com/) package manager

## Installation

1. **Clone the repository:**
   ```sh
   git clone https://github.com/AmitGarg1207/StackStories.git
   cd StackStories
   ```

2. **Install dependencies for both backend and frontend:**

   If your project has separate directories for the backend and frontend (e.g., `backend` and `client`):

   **For the Backend:**
   ```sh
   cd backend
   npm install
   ```

   **For the Frontend:**
   ```sh
   cd ../client
   npm install
   ```

## Running the Application

### Start the Frontend

1. Open a terminal and navigate to the `client` directory:
   ```sh
   cd client
   ```
2. Run the following command:
   ```sh
   npm start
   ```
   The frontend will start on **port 3000**.

### Start the Backend

1. Open another terminal and navigate to the `backend` directory:
   ```sh
   cd backend
   ```
2. Run the following command:
   ```sh
   node index.js
   ```
   The backend server will run on **port 4000**.

## Usage

- **User Authentication:** Register or log in to start posting.
- **Create Posts:** Share your stories by creating new blog posts.
- **Manage Posts:** Easily update or delete your posts.
- **Responsive Experience:** Navigate the platform seamlessly on any device.
